# 统一抽取结果弹窗设计

## 目标

把 `模板创建` 和 `数值直抽` 两个页面的抽取完成反馈统一成同一套节奏：

1. 当前字段转盘完整停下
2. 结果写入表格
3. 弹出一个小结果窗
4. 用户点击 `X` 或 `下一项`
5. 页面再切到下一个字段

这样可以避免“字段已经跳走，但用户还没看清当前抽中的是什么”的问题。

## 适用页面

### 模板创建

- 结果窗显示：
  - 字段
  - 最终值
  - 来源球员
  - 备注
- 如果不是最后一个字段，关闭弹窗后跳到下一项。
- 如果已经是最后一个字段，关闭弹窗后停留当前页，方便直接保存模板。

### 数值直抽

- 结果窗显示：
  - 字段
  - 最终值
  - 范围
  - 备注
- 如果不是最后一个字段，关闭弹窗后跳到下一项。
- 如果已经是最后一个字段，关闭弹窗后停留当前项。

## 交互规则

- 转盘旋转期间绝不切换字段。
- 弹窗打开时，背景内容保持可见，但不可交互。
- 弹窗支持：
  - 右上角关闭 `X`
  - 主按钮 `下一项`
  - 最后一项时主按钮文案改成 `完成`
