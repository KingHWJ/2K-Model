## 建模说明

- 当前字段布局优先满足“逐项抽字段”的创建体验，而不是一次性按大属性组整包抽取。
- 模板创建页的转盘每个扇区只代表一名球员，指针固定在顶部并停靠到扇区中心。
- 数值直抽页的转盘默认每个扇区只代表一个数字；`肩宽/体型` 这种字段会切换成离散文本选项。
- 模板创建页和数值直抽页都会在抽取完成后先弹出结果窗，关闭弹窗后才切到下一字段。
- 推荐模板的中文风格描述参考了 2K26 `Builder / archetype / scouting report` 的方向。
- 默认球员同时保留了英文别名和常用缩写，便于后续继续扩库或增加搜索入口。
- 推荐模板封面默认使用内置海报风占位图，也支持你上传自己的真实球星封面进行覆盖。
