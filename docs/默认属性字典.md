## 建模说明

- 默认标签参考的是 2K26 Builder / archetype / scouting report 的建模思路。
- 中文标签用于帮助快速识别球员模板风格，不要求和游戏原始文案逐字一致。
- 为了兼顾搜索体验，默认球员还内置了英文别名和常用缩写，例如 `MJ / KD / SGA / Joker / AD / Wemby`。
- 如果后续要扩展 `Badge / Takeover / Build Specialization`，建议作为新的属性组或新的模板扩展区单独加入。
