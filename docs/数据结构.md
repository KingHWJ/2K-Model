# 数据结构

## 本地存储键

- `2k26-fusion-builder:v2`

## 顶层状态 `AppState`

```ts
interface AppState {
  version: 2
  categories: AttributeCategory[]
  players: PlayerProfile[]
  recommendedTemplates: RecommendedTemplate[]
  tagDefinitions: TagDefinition[]
  numberFields: NumberDrawField[]
  numberSession: NumberDrawSession
  templates: SavedTemplate[]
  session: BuilderSession
}
```

## 属性分类 `AttributeCategory`

```ts
interface AttributeCategory {
  id: string
  name: string
  fields: Array<{
    key: string
    label: string
    defaultValue: number | string
  }>
}
```

用途：

- 定义字段所属分类。
- 生成创建页的字段流程条和结果表。

## 球员档案 `PlayerProfile`

```ts
interface PlayerProfile {
  id: string
  name: string
  position: string
  overall: number
  era: string
  tags: string[]
  aliases: string[]
  categories: Record<string, Record<string, number | string>>
}
```

用途：

- 保存球员基础信息。
- 保存各字段的真实值来源。
- 保存搜索别名和风格标签。

## 推荐模板 `RecommendedTemplate`

```ts
interface RecommendedTemplate {
  id: string
  name: string
  subtitle: string
  description: string
  featuredPlayers: string[]
  candidatePlayerIds: string[]
  tags: string[]
  defaultCover: string
  customCover: string | null
  featuredFieldIds: string[]
}
```

用途：

- 驱动首页模板推荐卡。
- 决定当前模板的候选球员池。
- 决定创建页字段抽取顺序。
- 保存默认封面和自定义封面。

## 标签定义 `TagDefinition`

```ts
interface TagDefinition {
  id: string
  label: string
  description: string
  featuredPlayers: string[]
  focusFields: string[]
}
```

用途：

- 驱动标签页展示。
- 解释不同模板风格重点。

## 创建会话 `BuilderSession`

```ts
interface BuilderSession {
  recommendedTemplateId: string | null
  templateName: string
  candidatePlayerIds: string[]
  fieldOrder: string[]
  currentFieldIndex: number
  fieldAssignments: Record<string, FieldAssignment>
  drawLog: DrawLogEntry[]
  createdAt: string
  updatedAt: string
}
```

## 字段抽取记录 `FieldAssignment`

```ts
interface FieldAssignment {
  fieldId: string
  playerId: string
  value: number | string
  createdAt: string
}
```

用途：

- 记录某个字段最终采用了哪名球员的值。
- 为即时结果卡和结果表提供数据来源。

## 数值直抽字段 `NumberDrawField`

```ts
interface NumberDrawField {
  id: string
  label: string
  min: number
  max: number
  defaultValue: number
  unit: string
  note: string
}
```

用途：

- 驱动 `数值直抽` 页左侧字段列表。
- 决定数字转盘的范围、默认值和单位。

## 数值直抽会话 `NumberDrawSession`

```ts
interface NumberDrawSession {
  activeFieldId: string
  results: Record<string, NumberDrawResult>
  createdAt: string
  updatedAt: string
}
```

## 数值直抽结果 `NumberDrawResult`

```ts
interface NumberDrawResult {
  fieldId: string
  value: number
  createdAt: string
}
```

用途：

- 记录每个内置字段最后一次抽出的数字结果。
- 支持刷新页面后恢复当前字段和历史抽取结果。

## 已保存模板 `SavedTemplate`

```ts
interface SavedTemplate {
  id: string
  name: string
  recommendedTemplateId: string | null
  candidatePlayerIds: string[]
  fieldOrder: string[]
  currentFieldIndex: number
  fieldAssignments: Record<string, FieldAssignment>
  coverImage: string
  summary: {
    overall: number
    tags: string[]
    recommendedTemplateName: string
    completedFields: number
    fieldResults: TemplateFieldResult[]
  }
  createdAt: string
  updatedAt: string
}
```

用途：

- 驱动 `我的模板` 页展示。
- 支持继续编辑时恢复回创建页。

## 字段结果 `TemplateFieldResult`

```ts
interface TemplateFieldResult {
  fieldId: string
  fieldLabel: string
  value: number | string
  playerName: string
  note: string
}
```

用途：

- 汇总展示每个字段的最终值、来源球员和备注。
- 为模板摘要和结果表提供直接展示数据。
