# 产品说明

## 页面结构

### 池子配置区

- 选择当前预设池。
- 调整候选人数。
- 编辑球员池：修改姓名、位置、总评、年代、标签和每个属性组里的字段值。
- 编辑属性池：修改属性组名称、字段名称、默认值，并支持新增/删除属性组与字段。
- 编辑标签库：新增和移除风格标签。

### 转盘抽选区

- `候选球员轮盘`：从当前预设池中随机抽出若干候选球员，默认去重。
- `属性大类轮盘`：用于承接每个属性组的随机来源。
- 每个属性组都支持：
  - 单独抽取来源球员
  - 从候选球员中手动切换来源
- `一键全抽` 会按当前属性组顺序一次性完成全部抽取。

### 融合结果区

- 显示当前模板名称、综合总评、位置画像、风格标签。
- 结果表格按属性组列出：
  - 属性大类
  - 来源球员
  - 当前字段预览
- 支持把当前融合结果保存到模板库。

### 模板库

- 展示所有已保存模板。
- 支持继续编辑、复制模板、删除模板。
- 保存后的模板会跟随浏览器本地数据一起持久化。

## 核心流程

1. 选择预设池并设置候选人数。
2. 抽取候选球员。
3. 对各属性组逐项抽取来源，或直接一键全抽。
4. 检查右侧结果表格和模板摘要。
5. 输入模板名并保存。
6. 在模板库中继续编辑、复制或删除。

## 交互规则

- 候选球员抽取默认不重复。
- 属性组抽取只从当前候选球员中选择来源。
- 手动切换来源时，也只允许从当前候选球员里选择。
- 编辑球员池或属性池后，预设池会自动同步引用范围。
- 模板保存后，如果当前是“继续编辑”某个模板，则会覆盖同一模板；否则会新增一条模板记录。

## 持久化策略

- 页面初始化时优先从 `localStorage` 恢复。
- 如果本地没有数据或数据损坏，则回退到默认内置数据。
- 每次编辑、抽取、保存模板后都会自动同步到本地。
