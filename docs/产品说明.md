# 产品说明

## 页面结构

### 首页

- 默认进入 `模板推荐首页`。
- 页面主视图展示 6 张推荐模板卡，例如：
  - 双向分卫模板
  - 高个投射核心
  - 推进攻框前锋
  - 禁区统治中锋
  - 高位策应中锋
  - 篮板防守工兵
- 每张卡包含：
  - 模板封面
  - 模板名
  - 一句话副标题
  - 风格描述
  - 代表球员
  - 核心标签
  - `基于此模板创建`
  - `上传模板封面`

### 创建模板页

- 点击首页卡片后进入 `创建模板` 页。
- 页面顶部展示当前模板横幅，延续首页封面和风格标签。
- 页面中部只保留一个 `单主转盘`：
  - 顶部固定指针
  - 扇区根据候选球员数量自动分布
  - 盘面球员名称保持横向显示，便于阅读
  - 抽取时会按目标扇区中心停靠，指针只会指向一个明确球员扇区
- 转盘右侧提供 `抽取当前字段` 按钮，转盘下方提供 `开始抽取` 按钮。
- 页面中部同时展示 `字段流程条`，例如：
  - 身高
  - 体重
  - 臂展
  - 肩宽/体型
  - 速度
  - 加速
  - 弹跳
  - 力量
  - 上篮
  - 扣篮
  - 中投
  - 三分
  - 控球
  - 传球
  - 外防
  - 内防
  - 抢断
  - 盖帽
  - 后场篮板
- 每次点击 `抽取当前字段` 后：
  - 转盘从当前候选球员里随机选一名来源球员
  - 当前字段立即生成最终值
  - 页面即时结果卡同步显示 `字段名称 / 最终值 / 来源球员 / 备注`
  - 结果表同步落一行，`最终值` 列会做高亮
  - 当前字段自动推进到下一项

### 数值直抽页

- `数值直抽` 是独立标签页，不依赖推荐模板。
- 页面左侧展示内置字段列表，字段包括：
  - 身高
  - 体重
  - 臂展
  - 肩宽/体型
  - 速度
  - 加速
  - 弹跳
  - 力量
  - 上篮
  - 扣篮
  - 近框终结
  - 中投
  - 三分
  - 罚球
  - 控球
  - 传球
  - 外防
  - 内防
  - 抢断
  - 盖帽
  - 后场篮板
- 页面中部展示 `数字饼图转盘`：
  - 每个数字都是一个独立扇区
  - 指针固定在顶部向下
  - 盘面按数值范围生成数字标签
  - 抽取时会停在某一个明确数字扇区上
- `肩宽/体型` 使用离散选项转盘：
  - 精瘦
  - 标准
  - 宽肩
  - 强壮
- 页面右侧展示即时结果和结果表，字段内容包括：
  - 默认值
  - 最终值
  - 范围
  - 备注
- 每次抽完当前字段后，页面会自动切换到下一个字段，减少手动点选。

### 标签页

- 标签页负责解释模板风格，不打断主创建流程。
- 每张标签卡会展示：
  - 标签名
  - 标签说明
  - 常见代表球员
  - 重点影响字段

### 我的模板

- 已保存模板会集中展示在 `我的模板` 页。
- 每张模板卡展示：
  - 保存时的封面
  - 模板名
  - 所属推荐模板
  - 已完成字段数量
  - 综合总评
  - 标签摘要
- 支持：
  - 继续编辑
  - 删除

## 核心流程

1. 进入首页，浏览推荐模板。
2. 选择一个模板方向，点击 `基于此模板创建`。
3. 在创建页按字段顺序逐项抽取。
4. 查看即时结果卡和字段结果表。
5. 输入模板名称并保存。
6. 在 `我的模板` 页继续编辑或删除。
7. 如需完全脱离模板，可切到 `数值直抽` 页直接按数字转盘决定字段结果。

## 交互规则

- 当前版本的候选球员由推荐模板预设提供。
- 字段抽取只会从当前模板对应的候选球员中选择来源。
- 保存模板前，至少要完成一个字段抽取。
- 如果当前是从模板库继续编辑，保存时会覆盖原模板；否则会新增一条模板记录。
- 首页模板封面优先级：
  - 优先显示用户上传封面
  - 没有自定义封面时显示系统默认封面
- 数值直抽默认范围：
  - 身高 `170-224 cm`
  - 体重 `70-170 kg`
  - 臂展 `175-250 cm`
  - 肩宽/体型 `精瘦 / 标准 / 宽肩 / 强壮`
  - 能力值默认 `65`，范围 `25-99`

## 持久化策略

- 页面初始化时优先从 `localStorage` 恢复最新状态。
- 如果本地数据损坏或版本不匹配，则回退到默认数据。
- 推荐模板自定义封面、当前创建会话、数值直抽结果、已保存模板都会自动持久化。
