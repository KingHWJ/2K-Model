# 2K26 球员融合模板工具

一个基于 `React + Vite + TypeScript` 构建的静态网页工具，用来从推荐模板出发，按 `身高 / 体重 / 臂展 / 肩宽 / 核心能力字段` 逐项抽取球员来源，快速拼出一套新的 2K26 风格模板。

## 当前版本亮点

- 首页先展示 `官方预设型模板推荐`，而不是直接进入复杂工作台。
- 点击 `基于此模板创建` 后，进入单个大转盘页面，按字段逐项抽取来源球员。
- 转盘会按真实目标扇区停靠，指针固定在顶部，盘面标签保持横向清晰可读。
- 创建模板页在转盘右侧和转盘下方各保留一个抽取入口，便于单手操作和桌面操作。
- 每次抽取后，结果表会立即写入 `字段 / 最终值 / 来源球员 / 备注`。
- 支持独立的 `数值直抽` 标签页，所有内置字段都可以直接通过数字转盘抽出最终值。
- 支持 `标签页` 查看模板风格说明。
- 支持 `我的模板` 查看已保存结果，并继续编辑。
- 推荐模板卡支持 `系统默认封面` 与 `用户自定义上传封面`。
- 数据全部保存在浏览器本地，无需后端。

## 本地启动

```bash
npm install
npm run dev -- --host 0.0.0.0
```

默认开发地址通常是 [http://localhost:5173](http://localhost:5173)。

## 说明与边界

- 当前默认球员数据优先满足“可演示、可玩、风格清晰”，不承诺与 NBA 2K26 游戏内字段逐项完全一致。
- 默认中文标签是基于 2K26 `Builder / archetype / scouting report` 思路做的贴近式建模，不是官方逐球员原文标签表。
- 数值直抽页内置了贴近 2K Builder 概念的默认范围，例如 `身高 170-224 cm`、`体重 70-170 kg`、`臂展 175-250 cm`，能力值默认基线为 `65`，范围为 `25-99`。
- 默认封面使用内置海报风占位图，功能上支持你替换成自己的真实球星封面。
- `Badge / Takeover / Build Specialization / Cap Breakers` 暂未进入当前主流程，后续可以继续扩展。
